## Quick Start

### Using Docker
```bash
# Build the image
docker build -t typescript-app .

# Run the container
docker run -p 8080:8080 typescript-app
```

### Using Docker Compose
```bash
# Start all services
docker-compose up -d

# View logs
docker-compose logs -f

# Stop services
docker-compose down
```
